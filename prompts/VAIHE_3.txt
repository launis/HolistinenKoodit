interface ArgumentaatioAnalyysi extends BaseJSON {
  toulmin_analyysi: Array<{
    vaite_id: string;
    claim: string;
    data: string;
    warrant: string;              // Oikeutus
    backing: string;              // Tuki (jos on)
  }>;
  kognitiivinen_taso: {
    bloom_taso: string;           // esim. "Analyze" tai "Evaluate"
    strateginen_syvyys: string;   // Arvio onko kyseessä syvällinen vai pinnallinen taso
  };
  walton_skeema: {
    tunnistettu_skeema: string;   // esim. "Argument from Expert Opinion"
    kriittiset_kysymykset: string[]; // Lista kysymyksistä vaiheelle 4
  };
}
KÄSKE: (Esimerkki oikeasta rakenteesta):
Sinun TÄYTYY noudattaa tätä rakennetta (huomaa tarkat avaimet!):
{
  "toulmin_analyysi": [
    {
      "vaite_id": "Väite-1",
      "claim": "...",
      "data": "...",
      "warrant": "...",
      "backing": "..."
    }
  ],
  "kognitiivinen_taso": {
    "bloom_taso": "Analyze",
    "strateginen_syvyys": "Syvällinen"
  },
  "walton_skeema": {
    "tunnistettu_skeema": "Argument from Expert Opinion",
    "kriittiset_kysymykset": [
      "Kysymys 1?"
    ]
  }
}
ÄLÄ käytä avaimia kuten "pisteytys", "taso" tai "kriteeri1" ellei niitä ole yllä mainittu. Käytä vain yllä olevia.
