Tiedosto: 7_falsifiointi_ja_etiikka.json

TypeScript


interface EtiikkaJaFakta extends BaseJSON {
  faktantarkistus_rfi: Array<{
    vaite: string;
    verifiointi_tulos: "Vahvistettu" | "Kumottu" | "Ei voitu vahvistaa";
    lahde_tai_paattely: string;
  }>;
  eettiset_havainnot: Array<{
    tyyppi: "Syrjintä" | "Haitallinen sisältö" | "Plagiointi" | "Ei havaittu";
    vakavuus: "Kriittinen" | "Varoitus" | "N/A";
    kuvaus: string;
  }>;
}
KÄSKE: (Esimerkki oikeasta rakenteesta):
Sinun TÄYTYY noudattaa tätä rakennetta (huomaa tarkat avaimet!):
{
  "faktantarkistus_rfi": [
    {
      "vaite": "...",
      "verifiointi_tulos": "Vahvistettu",
      "lahde_tai_paattely": "..."
    }
  ],
  "eettiset_havainnot": [
    {
      "tyyppi": "Ei havaittu",
      "vakavuus": "N/A",
      "kuvaus": "..."
    }
  ]
}
ÄLÄ käytä avaimia kuten "pisteytys", "taso" tai "kriteeri1" ellei niitä ole yllä mainittu. Käytä vain yllä olevia.
